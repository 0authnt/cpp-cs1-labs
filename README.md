# C++ CS1 Labs

This repository contains a cleaned and organized portfolio version of my **CS1 C++ laboratory work**.

It is intended to show progression in core C++ programming skills, including **procedural programming and introductory object-oriented programming (OOP)**.

## Purpose

This portfolio version is organized to demonstrate **domain knowledge of introductory C++** while clearly separating:

- my lab and exercise code
- instructor-provided lab materials
- scratch/sandbox code

Compiled binaries and editor-specific files were removed to keep the repository clean.

## Repository Structure

- `ClassExercises/` — early in-class exercises and small practice programs
- `Labs/` — lab submissions grouped by original lab folder
- `InstructorMaterials/` — assignment handouts, reference files, and instructor-provided support materials
- `Sandbox/` — experimental or scratch files that are not part of the main lab sequence

## Topics Demonstrated

Across these labs, the repository demonstrates work with:

- C++ source files and compilation
- variables, arithmetic, and console input/output
- conditionals and control flow
- loops and repeated computation
- functions and modular decomposition
- header/source separation
- arrays, strings, and text processing
- simple game and maze-style program structure
- classes and object-oriented design foundations
- list/vector-style abstractions in later labs

## Build Notes

Many folders contain standalone `.cpp` files or small multi-file programs. Build commands will vary by lab. Example:

```bash
g++ main.cpp -o program
./program
```


For multi-file programs, compile all required `.cpp` files together.
